What is LaunchMenu?

LaunchMenu is an Open Source, cross-platform application which brings utilities (applets) to your fingertips. Open it at a press of a button, with LaunchMenu's keyboard centric design!

Why LaunchMenu?

LaunchMenu is easy to use, free and allows you to increase your productivity. Customise the application to your needs with advanced theming and applet settings.

Extendable with Applets

Some applets come preinstalled but additional 3rd party applets will be available. LaunchMenu relies on applets to serve users, and doesn't limit developers in providing their desired user experience.

Keyboard centric

  • Select items

    Use the arrow keys to navigate the menu.

  • Execute item actions

    Hit

    Enter ↵
    to perform the primary action of the selected item.

  • Step in and out of menus

    Press the

    Tab ↹
    key to hop into a context menu and
    Escape
    to exit out of it again.

  • Customise any controls

    All controls are customisable to fit your needs! Join the ijkl-master race.

  • Native VIM keys mode

    LaunchMenu will have toggle-able custom keyboard layouts, including VIM as standard.

    • Search anywhere

  • All menus are searchable

    No matter where you are, simply start typing to start searching!

  • Searches are recursive

    Searches look through the selected menu as well as all sub-menus.

  • Filters

    Searches can use patterns to automatically filter results.

  • Fuzzy Search

    By default all searches use a fuzzy matching algorithm.

  • Prioritised searches

    All searches will be tracked and prioritised intelligently, bringing frequently used items closer to the top of the list.

    Tab ↹

  • Searching

  • Sub-menus

    ⇧ Shift

  • Shared context menu

    Opening a context menu will list actions from all the selected items. Actions executed will operate on all compatible items.

    • Customise

  • Settings

    Like all other items, settings can be searched, and altered easily.

  • Keyboard Shortcuts

    All keyboard shortcuts / controls can be fully customised to suit your needs.

  • Customise applets

    Like native LaunchMenu features, all applets - including 3rd party applets, will be customisable.

  • Settings portability

    Settings will be portable, allowing you to import and export settings.

  • Applets

  • Themes

    Colour and Style of the LaunchMenu app and applets will be customisable through themes.
